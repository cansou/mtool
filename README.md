# Mac 我的快捷键
###  ` 全局 ` 
### command + 空格: 打开全局搜索
<br>
### command + W : 关闭窗口,不关闭程序
<br>
### command + Q : 关闭窗口,退出程序
<br>
### command + A : 全选
<br>
### command + X : 剪切
<br>
### command + V : 粘贴
<br>
### 长按command, 点击文件,可以多选文件
<br>
###  ` Android Studio ` 
### option + return[回车] : 引入包,或者强制转换[Cast]提示
<br>
### command + X : 剪切整行
<br>
###  ` 微信 ` 
### control + L : 微信截图
